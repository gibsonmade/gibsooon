# Style Exploration

## Direction A: White Gallery, Black Type, Chrome Object

High-trust homepage with oversized type, asymmetric portrait/object composition, and restrained cosmic/chrome detail.

Best for launch because it balances premium credibility with experimentation.

## Direction B: Black Keynote, Bright Product Covers

Dark-mode-first experience with product mockups and glowing system diagrams.

Strong for work pages, but riskier as the main home experience if case study assets are not ready.

## Direction C: Editorial Notebook System

Magazine-like layouts using scans, notes, wireframes, and process artifacts.

Strong for About and Resources, but may need more personal source material from Gibson.

## Recommendation

Start with Direction A for the first build, then borrow Direction B for Work and Direction C for About/Resources.
