# Style Guide

## Style Name

Neo-Brutalist Cosmic Editorial.

## Visual Formula

- 60% Cosmic Editorial
- 20% Systems Thinking
- 10% Luxury Product Marketing
- 10% Gibson Personal Artifacts

## Mood

Futuristic, editorial, premium, intelligent, experimental, creative, trustworthy, high contrast, gallery-like.

## Core Rules

- Typography dominates.
- Every screen should contain one strong hero object.
- Never center everything.
- Use massive scale contrast.
- Use extreme negative space.
- Use small monospace labels.
- Use black, white, and restrained accent color.
- Client colors may influence accents but should not overtake the page.
- Product screenshots should feel like luxury advertisements.
- Major compositions should be strong enough to become a social thumbnail.
- The site should feel like a design conference keynote, not a normal portfolio.

## Color Direction

- Foundation: black, white, near-black, warm off-white.
- Accent: restrained cosmic blue, chrome silver, electric lime, or signal red.
- Avoid one-note palettes and generic purple-blue gradient dominance.

## Final Color Tokens

- `ink` / `#050505`: primary text, dark backgrounds, hard borders.
- `paper` / `#f7f4ee`: primary light background.
- `bone` / `#fffaf0`: warmer highlight surface for editorial breaks.
- `graphite` / `#1a1a1a`: secondary dark surface.
- `chrome` / `#c8d0d9`: metallic neutral for interface objects and visual treatments.
- `signal` / `#d9ff2f`: high-attention accent for CTAs, fit signals, and active roadmap states.
- `cosmic` / `#245cff`: digital/cosmic accent for focus states and glow.
- `solar` / `#ff5a1f`: restrained warm accent, useful for Home Depot-influenced moments.

Usage rule: black, white, and warm paper carry the interface. Accent colors should be signals, not wallpaper.

## Typography Direction

- Large editorial display type for page-defining statements.
- Small monospace labels for systems, metadata, status, and navigation cues.
- Refined readable body type for credibility.

## Type Scale

- Labels: `label-xs`, monospace, uppercase, generous letter spacing.
- Small body: `body-sm`, used for cards, metadata, and dense supporting copy.
- Body: `body-md`, used for readable paragraphs.
- Dek: `dek`, used for page summaries and major supporting statements.
- Display medium: `display-md`, used for section-level editorial heads.
- Display large: `display-lg`, used for page-defining statements.

Rule: display type should create hierarchy, not fight the content. Inside compact cards and panels, use smaller headings with tighter line lengths.

## Interaction Direction

- Confident hover states.
- Keyboard-visible focus states.
- Motion should feel intentional and editorial, not decorative.
